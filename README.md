# NCT Delhi UHI and Air Pollution Analysis

This repository contains data, code, results, and the final report for the NCT Delhi Urban Heat Island (UHI) and air pollution analysis.

## Repository Contents

### `2020_to_2024_csv_files_13_observed_centers`
Contains cleaned/compiled CSV files for the 13 selected observation stations used in the 2020–2024 analysis.

### `Air_pollution_results_for_year_2023_and_2024`
Contains processed air pollution result files for 2023 and 2024, including pollutant summaries and correlation-related outputs.

### `GEE_Java_Script_codes`
Contains Google Earth Engine JavaScript codes used for mask creation, Landsat/MODIS extraction, LST, NDVI, and SUHI-related processing.

### `GEE_extracted_LST_NDVI_csv`
Contains CSV files exported from Google Earth Engine for LST, NDVI, and related satellite-derived outputs.

### `caaqms_station_csv`
Contains CAAQMS station-level air quality CSV files used for pollutant analysis.

### `UHI_and_Air_pollution_Analysis_of_NCT_Delhi.pdf`
Final project report PDF.
